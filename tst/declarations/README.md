# Century Ada Compiler

There are a fair number test cases in this folder.  As this note is being written during the unit testing of Chapter 3 code, there are several items which can be expected to parse and several which will not parse properly.  The root of this is that several rules have dependencies on other things which are presented in later chapters.  `expression` is a prime example of this situation, which is presented in Chapter 4.

Therefore, I wanted to break this up into a list which will identify when each test case herein can be expected to parse so that I (and you as an outside observer) know what to expect.


## Command Line

The command line to process any of these test cases is:

`bin/ada-cc types tst/declarations/<test-case>.ada`
